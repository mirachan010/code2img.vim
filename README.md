# code2img
The Vim plugin that can generate image of source code.

## Usage
```vim
" generate image of file
:Code2img

" generate image of selected range
:'<,'>Code2img
```

## Requirements
- [code2img](https://github.com/skanehira/code2img)
- xclip(linux only)

## Installation
e.g dein.vim

```toml
[[plugin]]
repo = 'github.com/skanehira/code2img.vim'
```
